# Bogotá Artística & Cultural

Plataforma web interactiva para la navegación y localización de circuitos culturales en Bogotá. El proyecto nace como una guía digital para centralizar información dispersa sobre espacios artísticos, con énfasis inicial en la localidad de Chapinero.

## Propósito

Facilitar el acceso a la oferta cultural de la ciudad mediante un mapa navegable que permite ubicar, filtrar y explorar espacios relacionados con cine, teatro, museos, galerías, danza, música y rutas temáticas.

## Características

- Mapa interactivo de Bogotá con marcadores por lugar cultural.
- Búsqueda de espacios por nombre, zona o etiquetas.
- Filtros por categoría artística.
- Rutas temáticas como Ruta Cinéfila, Ruta Cultural, Ruta Escénica y Ruta Urbana.
- Panel lateral con información práctica de cada lugar: descripción, horarios, acceso, precio, contacto, etiquetas y enlaces.
- Diseño visual orientado a una experiencia inmersiva y de lectura rápida.

## Contexto del proyecto

Este desarrollo se inspira en la propuesta académica “Guía del turista del arte”, que plantea una herramienta digital para reducir la fragmentación de la información cultural en Bogotá y fortalecer la visibilidad de espacios artísticos independientes y comerciales.

La fase inicial se concentra en Chapinero, por su alta densidad cultural y su valor como punto de partida para validar el modelo de circuitos culturales antes de escalar a otras zonas de la ciudad.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Leaflet.js
- OpenStreetMap / capas cartográficas de Leaflet

## Estructura del proyecto

- `index.html`: estructura principal de la interfaz.
- `css/styles.css`: estilos visuales y adaptaciones responsive.
- `js/app.js`: lógica del mapa, filtros, rutas y panel lateral.
- `data/lugares.js`: base de datos de lugares culturales.

## Cómo ejecutar el proyecto

1. Clona o descarga el repositorio.
2. Abre la carpeta del proyecto en VS Code.
3. Ejecuta la vista previa con un servidor local, por ejemplo Live Server.
4. Abre `index.html` en el navegador.
